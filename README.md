# 💼 Sujit Malik - Developer Portfolio

🌐 **Live Website**: [https://sujitportfo.netlify.app/](https://sujitportfo.netlify.app/)

- **Backend (optional for future integration)**: Node.js, Express.js
- **Database (future-ready)**: MongoDB
